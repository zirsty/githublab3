# Проект информационной системы «Учёт сеансов в кинотеатре»

## 1 Постановка задачи

### 1.1 Предметная область

Система автоматизирует управление киносеансами в кинотеатре. Она учитывает:

- **Фильмы**: название, жанр, длительность, возрастное ограничение  
- **Расписание сеансов**: фильм, дата, время начала, зал  
- **Залы кинотеатра**: номер, вместимость, тип  
- **Билеты**: ряд, место, статус (**свободен**, **забронирован**, **продан**)

---

### 1.2 Цели системы

- Упрощение формирования и изменения расписания  
- Автоматизация продажи и бронирования билетов  
- Исключение ошибок, связанных с ручным учётом  
- Получение отчётов о посещаемости и доходах  
**Польза:**  
Система делает работу кинотеатра эффективнее, а зрителям предоставляет удобное получение информации о фильмах и сеансах.

---

### 1.3 Пользователи системы и их роли

| Роль | Функции |
|------|--------|
| **Администратор кинотеатра** | Управляет списком фильмов, залами, формирует расписание |
| **Кассир** | Выполняет бронирование и продажу билетов |
| **Зритель (при онлайн-доступе)** | Просматривает расписание, бронирует билеты |

---

### 1.4 Основные функции системы

- Ведение базы фильмов и залов  
- Формирование расписания сеансов  
- Просмотр расписания  
- Бронирование и продажа билетов  

---

### 1.5 Архитектура и способы реализации

Для реализации используется **трёхуровневая архитектура**:
- **Presentation Layer (интерфейс)** — отображение расписания, работа кассира/зрителя  
- **Business Logic Layer (бизнес-логика)** — проверка правил: свободные места, возрастное ограничение, статус билета  
- **Data Layer (данные)** — база данных фильмов, залов, сеансов и билетов  

---

### 1.6 Описание сущностей

| Сущность | Поля |
|--------|------|
| **Фильм** | `id`, `название`, `жанр`, `длительность_мин`, `возрастное_ограничение`, `описание` |
| **Зал** | `id`, `номер`, `вместимость` |
| **Сеанс** | `id`, `фильм_id`, `зал_id`, `дата`, `время_начала`, `время_окончания`, `цена_базовая` |
| **Билет** | `id`, `сеанс_id`, `ряд`, `место`, `посетитель_id`, `дата_продажи`, `цена_итоговая` |
| **Посетитель** | `id`, `фамилия`, `имя`, `телефон` |
| **Админ** | `id`, `имя`, `логин`, `пароль` |
